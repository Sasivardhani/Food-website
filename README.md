# Food-website
A sleek, responsive landing page for a culinary brand built with pure HTML and CSS. This project focuses on modern UI/UX principles, featuring a mouth-watering hero section, a categorized food menu, and a functional contact form.
A clean, responsive, and modern landing page template designed for restaurants. This project features a categorized food exploration section, a functional search bar, and a sleek navigation menu.

🚀 Features
Responsive Design: Optimized for various screen sizes using a mobile-first approach.

Food Search: A dedicated search section for users to find their favorite dishes.

Category Highlights: Visual cards for food categories like Pizza, Burgers, and Momos.

Social Integration: Links to Facebook, Instagram, and Twitter using Fluent icons.

Clean Code: Organized HTML5 structure with semantic sections for easy customization.

🛠️ Getting Started
Prerequisites
You only need a modern web browser (Chrome, Firefox, Edge, or Safari) to view this project.

Installation
Clone or Download this repository.

Ensure your folder structure matches the Project Structure section above, specifically the images folder.

Open index.html in your browser.

🖌️ Customization
Logo: Replace images/logo.png with your own branding.

Categories: Add more food items by duplicating the <div class="box-3"> blocks in the Categories section.

Styling: Update the style.css file to change the primary colors (btn-primary) or font styles.
